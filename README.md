# 🚗 Garage Billing System

## 📌 Project Description
A Java console-based application built using OOP concepts and Java Collections Framework to manage car service billing. This system helps in managing customers, services, and generating bills efficiently.

---

## 🚀 Features
- Add Customer Details
- View Available Services
- Generate Bill
- Service Management System

---

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)
- Java Collections Framework

---

## ▶️ How to Run
1. Open project in any Java IDE (IntelliJ / Eclipse)
2. Run the Main class
3. Follow on-screen instructions
